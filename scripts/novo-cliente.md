# Checklist — Novo Cliente White Label

## Informações necessárias
- [ ] Nome do sistema
- [ ] Domínio do cliente
- [ ] Logo (PNG transparente, 400x400px)
- [ ] Cores (primária e secundária)
- [ ] Chave PIX para recebimentos
- [ ] Email do admin
- [ ] Senha do admin

## Configuração técnica
- [ ] Criar projeto Supabase
  - Anotar URL e keys
- [ ] Executar migrations SQL
  - Rodar supabase/schema.sql
  - Rodar supabase/seed.sql
- [ ] Criar projeto Vercel
  - Conectar ao repositório GitHub
  - Configurar variáveis de ambiente
  - Configurar domínio
- [ ] Criar usuário admin inicial
  - Email: fornecido pelo cliente
  - Role: admin
- [ ] Configurar white label inicial
  - Logo
  - Cores
  - Nome do sistema
  - Slogan

## Entrega ao cliente
- [ ] Enviar credenciais de acesso
- [ ] Enviar tutorial de uso
- [ ] Agendar call de onboarding
- [ ] Documentar no DEPLOY.md
